## 文章结构

`desc`：长度为2的数组，[0]为声母读音（对于零声母为空字符串），[1]为韵母读音。该字段用于对文字注音显示，对标点符号这类无需注音的字符，省略此键

`keys`：长度为2的数组，[0]是声母对应的双拼按键（对于零声母遵守各输入法零声母方案的规则），[1]是韵母对应的双拼按键。

注：当`desc`字段被省略时，同时省略`dmap`字段。

```json
{
    "article": [
        {
            "word": "偶",
            "desc": ["", "ǒu"],
            "keys": ["o", "u"]        // 小鹤["o", "u"] 微软["o", "b"]
        },
        {
            "word": "昂",
            "desc": ["", "áng"],
            "keys": ["a", "h"]        // 小鹤["a", "h"] 微软["o", "h"]
        },
        {
            "word": "天",
            "desc": ["t", "iān"],
            "keys": ["t", "m"]
        },
        {
            "word": "只",
            "desc": ["zh", "ǐ"],
            "keys": ["v", "i"]
        },
        {
            "word": "啊",
            "desc": ["", "a"],
            "keys": ["a", "a"]        // 小鹤["a", "a"] 微软["o", "a"]
        },
        {
            "word": "！"
        }
    ]
}
```
